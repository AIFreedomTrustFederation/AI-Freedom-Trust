# AetherCore Test 001: Trust Density and COVID Shock Recovery

This project tests whether World Values Survey Wave 7 trust variables improve prediction of COVID-19 shock-response outcomes after controlling for conventional country-level factors from Our World in Data.

## Structure

- `data_raw/`: downloaded or manually placed source data
- `data_processed/`: cleaned country-level merged datasets
- `scripts/`: executable analysis code
- `outputs/`: regression tables and model summaries
- `figures/`: plots generated by the analysis
- `report/`: plain-English findings report

## Data Notes

The OWID COVID dataset is downloaded automatically from:

`https://covid.ourworldindata.org/data/owid-covid-data.csv`

World Values Survey Wave 7 may require free registration and license acceptance through the WVS website. The script first attempts a public OSF mirror of the official Wave 7 CSV (`https://osf.io/36dgb/download`). If automatic download is blocked, manually download the Wave 7 country-pooled file from:

`https://www.worldvaluessurvey.org/WVSDocumentationWV7.jsp`

Place the downloaded `.csv`, `.dta`, `.sav`, `.xlsx`, or `.zip` file in `data_raw/`, then rerun:

```powershell
python scripts\run_analysis.py
```

The script auto-detects common WVS Wave 7 variable names and writes an audit file describing which trust columns were used.
